# jarvis
для использования Джарвиса установить vosk https://alphacephei.com/vosk/models/vosk-model-ru-0.42.zip https://alphacephei.com/vosk/models/vosk-model-small-ru-0.22.zip

установите все необходимые библиотеки они в файле библиотеки.txt

в папке установите голос RHVoice-voice-Russian-Vitaliy-v4.0.2009.14-setup.exe

для использования режима общения скачайте олламу https://ollama.com/download/OllamaSetup.exe

запустите олламу и скачайте модель deepseek-r1:8b

запустите файл jarvis_launcher

настройте Джарвиса под себя и для того чтобы все изменения применились до конца перезапустить Джарвиса
