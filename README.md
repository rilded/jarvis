J.A.R.V.I.S. - История обновлений

Версия 9.0
Новый голосовой движок: Голос заменен на кастомный, записанный и обработанный программистом-блогером, вдохновившим проект. 
Качество и узнаваемость голоса значительно улучшены.
Расширение графического интерфейса: Добавлен удобный и интуитивно понятный конструктор команд. 
Пользователи могут создавать собственные команды и последовательности действий через GUI.


Версия 8.4
Глобальное расширение функционала: Добавлен большой набор новых команд и возможностей.
Публикация проекта: Первый выкладка исходного кода в публичный репозиторий на GitHub.


Версия 8.2
Оптимизация и юмор: Исправление множества ошибок (hotfix). 
Добавлена система случайных шуток и острот от J.A.R.V.I.S. для более живого общения.


Версия 8.0
Режим диалога: Интеграция нейросетевой модели (ИИ) для ведения свободных диалогов и ответов на общие вопросы, помимо стандартных команд.


Версия 7.0
Визуальное обновление: Полноценный графический интерфейс (GUI) с окнами, кнопками и визуальными настройками. 
Управление программой вышло за пределы командной строки.


Версия 6.0
Масштабная отладка: Огромное обновление, посвященное исправлению накопившихся ошибок, 
повышению стабильности и оптимизации кодовой базы.


Версия 5.0
Улучшение голоса: Переход с базового синтезатора на низкий мужской голос с улучшенным качеством и естественностью звучания.


Версия 4.0
Расширение языкового ядра: Значительно увеличено количество поддерживаемых команд и их вариаций для распознавания. 
Система стала лучше понимать естественную речь.


Версия 3.0 
Постоянное слушание и активация по имени:
Переименование проекта в J.A.R.V.I.S.
Реализация режима фонового прослушивания.
Активация ассистента по ключевой фразе "Джарвис".
Добавлен базовый синтез речи для ответов.


Версия 2.0
Голосовой ввод: Интеграция библиотеки Vosk для распознавания речи. Появилась возможность давать команды голосом (режим запускается по команде).


Версия 1.0
Рождение проекта: Текстовый интерфейс в командной строке (CMD). Реализован базовый набор команд и скриптов. Заложен фундамент системы.
